AI/Automation

Using Make.com Automation

About the Project

This project was designed to automate the process of summarizing news articles and creating social media content in bulk. My friend requested an automated system that could streamline this workflow, and I built it using Make.com.

Project Overview

The automation involves reading news articles, summarizing them using ChatGPT, and saving the results into Google Docs. These summaries are then used to create social media cards in Canva. The workflow is efficient and designed for bulk processing, making it ideal for content creators or marketers.

As the UX designer, I handled the app's design and prototyping, ensuring accessibility through high-contrast colors, voice-activated search, and intuitive navigation. This project aimed to create a space where anyone can feel like a chef while fostering a sense of community through recipe sharing.

Workflow Process

  • - Step 1: Use HTTP to fetch 10 news articles from a webpage.
  • - Step 2: Convert the articles into plain text using a parser.
  • - Step 3: Summarize the articles using ChatGPT API.
  • - Step 4: Format the summaries into JSON format.
  • - Step 5: Save the summaries into a Google Doc.

Outcomes

  • - The automated workflow successfully reduced manual effort by handling repetitive tasks like summarizing news and creating social media content in bulk.
  • - The integration of ChatGPT API ensured accurate and concise summaries, improving the quality of the content.
  • - Using Make.com for automation provided a seamless and efficient process, saving significant time
  • - The system’s ability to generate Google Docs and prepare content for Canva streamlined the creation of visually appealing social media cards.
  • - Overall, the project demonstrated how automation can enhance productivity and deliver consistent, high-quality results.

Conclusion

This project highlights the potential of combining AI tools like ChatGPT with powerful automation platforms such as Make.com. By addressing a real-world need, it showcases how technology can simplify complex workflows and allow users to focus on creativity rather than repetitive tasks. The success of this project reflects the value of automation in improving efficiency, accuracy, and scalability for content creation, making it a valuable solution for modern-day challenges.
